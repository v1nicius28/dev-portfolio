Esta é a versão full-stack do sistema de cadastro de pets. O backend, construído em Java 17 e Spring Boot, utiliza PostgreSQL (Neon) para persistência e está hospedado no Render. A interface web, desenvolvida com React e TailwindCSS, está hospedada no Vercel e permite o CRUD (Cadastro, Consulta, Edição e Remoção) completo dos pets. O projeto é containerizado com Docker.
